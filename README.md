# Arkitektur

Backend for **Arkitektur**, a modern architecture services UI platform.  
Built with .NET 9 using layered architecture, Repository & Result patterns, DI, Mapster, and Scrutor.

---

## Tech Stack

| Category | Tools / Libraries |
|----------|-----------------|
| **Backend** | C#, .NET 9 |
| **ORM / Data** | EF Core (Code-First), Repository Pattern |
| **Architecture / Patterns** | Layered Architecture, DI, Result Pattern |
| **Mapping / Utilities** | Mapster, Scrutor |
| **Authentication (Planned)** | JWT |
| **Storage (Planned)** | AWS S3 Bucket |
